# ENEC

ENEC is a lossless encoder designed for AI models, such as Large Language Models (LLMs), to enable efficient storage and high-speed inference on Ascend Neural Processing Units (NPUs). It supports models in FP16, BF16, and FP32 formats.

Unlike many existing lossless compression solutions tailored for CPUs and GPUs, ENEC is the first open-source lossless compression framework specifically optimized for NPUs. This fills a critical gap as NPUs become increasingly dominant in handling modern AI workloads.
